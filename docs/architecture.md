```
┌──────────────────────────────────────────────────────────┐
```

## 데이터 흐름

1. 사용자가 질문 입력 + 국가/비교항목 선택
2. FastAPI → Orchestrator Agent로 전달
3. Orchestrator가 질의 분석 후 필요한 에이전트 결정
4. 각 에이전트가 병렬로 작업 수행:
   - Research: 지식베이스에서 관련 정보 검색 → LLM으로 정리
   - Analysis: 비교 항목별 국가 전략 비교 → JSON 구조화
   - Insight: 시사점 및 한국 정책 제언 생성
   - Reference: 관련 참고문헌 및 링크 수집
5. Orchestrator가 모든 결과를 통합하여 최종 응답 생성
6. 구조화된 응답 (분석 텍스트 + 비교 테이블 + 레퍼런스) 반환
