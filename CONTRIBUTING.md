# 기여 안내 (Contributing)

이 프로젝트는 AI for Science 국가전략 비교의 정확성과 최신성을 높이기 위해 외부 피드백을 환영합니다.

## 피드백 유형

### 1. 사실 오류 제보
- 국가 전략 요약, 비교 내용, 데이터에서 사실과 다른 부분을 발견하셨다면 알려주세요.
- 가능하면 공식 출처(문서명, URL, 발행기관)를 함께 제시해 주시면 반영이 빠릅니다.

### 2. 최신 자료 제안
- 각국의 새로운 정책 발표, 행동계획 업데이트, 프로젝트 현황 변경이 있다면 알려주세요.
- 공식 발표자료 링크를 포함해 주시면 검증 후 반영합니다.

### 3. 전략 해석 의견
- 국가별 전략 유형 분류, 포지셔닝, 비교 해석에 대한 의견은 Discussions에서 나눌 수 있습니다.
- 근거 기반 의견을 환영합니다.

### 4. 문구 수정 제안
- 서비스에 노출되는 문구의 표현, 톤, 정확성에 대한 수정 제안을 받습니다.

## 제보 방법

- **Issue**: 사실 오류, 최신 자료, 문구 수정
- **Discussion**: 전략 해석, 방법론, 방향성 토론
- **Pull Request**: 직접 수정 제안 (데이터/문서 중심)

## 반영 절차

1. Issue 또는 Discussion 등록
2. 관리자 검토 (근거 확인)
3. 내부 검증 워크플로우 실행 (Critic → Router → Reviser)
4. 수정안 반영 및 공개

## 기여 원칙

- 공식 출처 기반의 제보를 우선합니다.
- 특정 국가에 대한 근거 없는 평가는 반영하지 않습니다.
- 사실과 해석을 구분하여 기여해 주세요.
