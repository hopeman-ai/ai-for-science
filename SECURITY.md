# 보안 정책 (Security Policy)

## 원칙

이 저장소에는 **API 키, 토큰, 비밀번호, 개인정보 등 민감정보가 포함되지 않습니다.**

## 공개 저장소에 올리지 않는 정보

- OpenAI API 키 또는 기타 서비스 토큰
- `.env` 파일 (`.env.example`만 포함)
- 데이터베이스 접속정보
- 내부 서버 주소 또는 운영 URL
- 개인 식별정보 (PII)
- 인증·인가 관련 설정값

## 보안 문제 제보

보안 취약점을 발견하셨다면:

1. **공개 Issue로 등록하지 마세요.**
2. 저장소 관리자에게 직접 연락해 주세요.
3. 제보 내용: 발견 위치, 재현 방법, 예상 영향

## 보안 점검 절차

이 저장소는 공개 전 아래 절차를 거칩니다.

- `.env` 파일 미포함 확인
- 하드코딩된 credential 스캔
- Git 커밋 이력 내 민감정보 검사
- `.gitignore` 설정 검증
